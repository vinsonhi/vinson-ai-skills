# 美股股票早报说明

## 目标

给用户的自选股票生成一份中文日报，覆盖价格变化、关键驱动和重要新闻。

如果 `instructions/us_stocks_watchlist_default.txt` 为空，说明这是一份可复用模板版本，默认不内置任何个人观察名单；此时应优先使用用户显式提供的股票列表。

## 必带字段

每只股票都必须包含：

1. 最新股价
2. 相比昨收的绝对变动和百分比
3. 影响变动的关键因素
4. 1-3 条重要新闻链接
5. 一句话判断
6. 顶部的价格时间口径说明（`as of` 时间）

## 数据源优先级

1. `web.finance`
2. 公司 IR / 财报 / 官方博客 / 官方新闻稿
3. Reuters
4. 其他权威媒体

## 输出格式

```markdown
## NVDA
- **最新股价**：$184.77
- **较昨天**：+1.16%
- **关键因素**：GTC 预期、产品发布、订单、生态合作。
- **重要新闻**：
  - [NVIDIA GTC 2026 官方预告](https://example.com)
- **我的判断**：会前预热行情仍在。
```

## 注意

- 默认使用“生成时最新价快照”，不要自动混成“上一交易日收盘后版本”。
- 同一份报告里的所有股票必须来自同一轮行情快照，不能混用多个时间点。
- 如果用户明确要求收盘版，再说明这份报告对应的是美东哪个交易日。
- 按北京时间处理周末与周一：
  - 周六运行：默认允许使用刚结束的周五收盘口径，做统一价格对比。
  - 周日运行：默认不要做价格对比，只写公司/行业信息、官方新闻、观察点和已知催化。
  - 周一运行：默认写最近一个美股交易日的盘后价格波动和信息更新，不能伪装成新的周一盘中价格。
- 周日和周一如果采用最近一个交易日收盘口径，顶部必须明确写清对应的是哪个美东交易日。
- 如果某只股票缺少近 24 小时新闻，不要编造，直接写“当天未见新的高可信催化，股价主要受板块和市场情绪驱动”。
- 高可信来源优先于数量。
