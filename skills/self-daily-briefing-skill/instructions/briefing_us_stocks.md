# 美股股票早报说明

## 目标

给用户的自选股票生成一份中文日报，覆盖价格变化、关键驱动和重要新闻。

## 必带字段

每只股票都必须包含：

1. 最新股价
2. 相比昨收的绝对变动和百分比
3. 影响变动的关键因素
4. 1-3 条重要新闻链接
5. 一句话判断

## 数据源优先级

1. `web.finance`
2. 公司 IR / 财报 / 官方博客 / 官方新闻稿
3. Reuters
4. 其他权威媒体

## 输出格式

```markdown
## NVDA
- **最新股价**：$184.77
- **较昨天**：+1.16%
- **关键因素**：GTC 预期、产品发布、订单、生态合作。
- **重要新闻**：
  - [NVIDIA GTC 2026 官方预告](https://example.com)
- **我的判断**：会前预热行情仍在。
```

## 注意

- 如果用户在中国时区，明确说明这份报告对应的是美东哪个交易日。
- 如果某只股票缺少近 24 小时新闻，不要编造，直接写“当天未见新的高可信催化，股价主要受板块和市场情绪驱动”。
- 高可信来源优先于数量。
