# 自用日报菜单

请回复序号或直接回复类型名称。

| # | 日报 | 说明 |
|---|---|---|
| 1 | 综合早报 | 科技、创投、社交、市场热点 |
| 2 | 财经早报 | 宏观、市场、板块、加密 |
| 3 | 科技早报 | AI、开发工具、创业产品 |
| 4 | AI 深度日报 | 论文、newsletter、行业观点 |
| 5 | 美股股票早报 | 自选股票价格、涨跌、驱动、重要新闻 |
| 6 | 合并版中文日报 | 把综合/财经/科技/AI 深度/美股合并到一个 Markdown 文件 |

### 美股股票早报默认观察名单

可复用版本默认留空；如果没有默认名单，必须先向用户要股票列表，例如：
`美股股票早报：TICKER_A,TICKER_B`

### 自由输入示例

- `综合早报`
- `美股股票早报`
- `美股股票早报：TICKER_A,TICKER_B`
- `科技早报 + AI 深度日报`
- `合并版中文日报`
